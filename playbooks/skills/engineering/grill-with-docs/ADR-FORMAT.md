# ADR Format

The ADR format is defined canonically in
[`playbooks/conventions/adr-convention.md`](../../../conventions/adr-convention.md).

In short: ADRs live in `docs/adr/` as `NNNN-slug.md`, the body is 1–3 sentences (context, decision,
why), and you offer one only when the decision is hard to reverse, surprising without context, and the
result of a real trade-off. See the convention for numbering, optional sections, and the full
offer-test.
